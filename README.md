# Invoice Hub

Invoice Hub is a sleek and intuitive invoice management platform designed to streamline the creation and tracking of invoices.

## Run the app

1. Install dependencies

```bash
  npm install
```

2. Start development server

```bash
  npm run dev
```

## Run the test

- Start the test cases

  ```bash
   npm run test
  ```

## Run the lint

1. Start the lint

   ```bash
    npm run lint
   ```
